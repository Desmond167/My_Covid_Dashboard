Introduction
We all know that Covid-19 has turned out to be a global pandemic and has put many innocent lives at risk. This is a situation in which we all need to come together and help each other. We all are doing our parts. I tried helping people know the condition in our country INDIA with the help of this Dashboard. People with no technical knowledge can also understand the conditin after checking this Dashboard.
Is this website correct?
All the data in this page, is fetched from the Government API or in easier terms from the Government Data-Base. So, each and every Graph in this website is 100% correct and no misguidings are present. The graphs are refreshed as soon as the Government Data-Base is updated. You can even find the 'Last-Updated' label in the graphs and tables.
Why did you create this website?
In this situation of global pandemic, I tried to use my skills to produce something usefull. I used my Data Visualization skills and Web Development knowledge to create this website. This is nothing more than a project. Here I tried to apply my skills so that it is easier for people to get an idea of the situation. This website is still in its beta version and I intend to release more versions. If any faults are present, then please do let me know. You can mail me from the About Me page.
